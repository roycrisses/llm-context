## 2025-05-14 - Sensible CLI Defaults and Feedback
**Learning:** CLI tools should provide sensible defaults for arguments (like defaulting to the current directory) to reduce typing overhead, and offer immediate feedback on the impact of actions (like displaying token counts after copying/saving context) to help users manage their LLM budgets.
**Action:** Default the DIRECTORY argument to '.' and display total token counts in success messages for clipboard and file operations.
